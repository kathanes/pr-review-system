# PR Review System

## 🚀 Overview

A scalable Pull Request Review System that simulates core features of modern code review platforms like GitHub.

## 🔥 Features

* Pull Request creation and tracking
* Review workflows (Approve / Request Changes)
* Inline comments on code
* File-level diff storage
* Pagination and filtering

## 🏗️ Tech Stack

* Java, Spring Boot
* Oracle Database
* JPA (Hibernate)
* Swagger (API Docs)
